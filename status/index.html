STATIC RELEASE RECORDS

Public release inventory

These are static publication records, not runtime monitors. They intentionally have no per-row event timestamp; use the current release manifest for source-commit and artifact-hash evidence.

Static release records are not live checks. Their validation scope and limitations remain visible without implying a current event.
